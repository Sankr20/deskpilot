# Changelog

## 0.1.0 (2026-01-25)

Initial public release.

Highlights:
- Windows desktop automation engine with DPI-safe coordinate normalization
- Locator system: point, region, template matching, and OCR-contains verification
- Actions API with step-scoped run folders and rich failure artifacts
- Recorder mode that generates runnable Java tests
- CLI distribution packaging (`deskpilot-dist.zip`)
